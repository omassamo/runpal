What if running was different?

Run Pal is a service making it easier for people to find Run Pals and start running together. Input your running preferences to create a profile and get matched with other runners. As Run Pal is still under development we would love it if you give your feedback after trying the service. Should you have any questions about the project don't hestitate to contact us. If you're interested in updates about the project follow us on twitter.
